Main Personal Projects

  • Aviation Accident Analysis:


    This interactive dashboard was developed using Python, Dash, and Google Cloud Platform (GCP). It processes and visualizes a large dataset of aviation accidents to help identify patterns and improve aviation safety. Users can explore accident trends across different aircraft types, geographic regions, and causes (e.g., pilot error, mechanical failure).

    The dashboard provides filters for:
    • Location – See which countries have the highest number of incidents.
    • Year – Observe trends over time (1974 - 2022) and examine high-risk periods.
    • Engine Type – Compare the frequency of accidents across Turbo Prop, Turbo Shaft, and other aircraft engine categories.
    • Weather Conditions – Evaluate how weather may contribute to accident likelihood.
    Behind the scenes, the project leverages Pandas for data wrangling, Plotly for visualization, and Dash for a responsive web interface. It’s deployed via Google Cloud Run for scalability and performance.

  • Cloud Computing Car Prediction Project:

The model was trained on a robust dataset and fine-tuned to improve classification accuracy across vehicle types and configurations.

    The predictive pipeline includes:
    • Model Training & Deployment – Utilized AWS SageMaker for orchestrating end-to-end machine learning workflows.
    • Deep Learning – Developed neural network models using TensorFlow and PyTorch to enhance prediction accuracy.
    • Feature Engineering – Applied preprocessing techniques to optimize model input quality and relevance.
    Behind the scenes, the project uses Python for development, PyTorch and TensorFlow for modeling, and AWS S3 for secure data storage and retrieval during training and evaluation.

  • Pyjack – Command-Line Blackjack Game:


    This interactive command-line game replicates the classic casino game of Blackjack using Python. Built as part of a foundational programming project, Pyjack lets players go head-to-head against a virtual dealer while managing bets and making real-time decisions to hit or stand.

    The game includes:
    • Gameplay Logic – Simulates shuffling, dealing, score calculation, bust conditions, and dealer behavior.
    • Betting System – Players start with a cash reserve and place bets each round, encouraging strategic play.
    • Difficulty Levels – Choose between "easy", "medium", or "hard" modes to adjust challenge level.
    • Simple CLI Commands – Use intuitive commands like b (begin), h (hit), s (stand), and q (quit).
    Behind the scenes, the game is entirely powered by Python and can be run either locally or directly through pip installation using pip install pyjack-foundations followed by pyjack_foundations.
